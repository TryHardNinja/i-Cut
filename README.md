i-Cut 是一个快速成形，全终端适配的CSS框架，采用 [Flexbox](https://developer.mozilla.org/zh-CN/docs/Web/CSS/flex) 与 [Grid-Layout](https://developer.mozilla.org/zh-CN/docs/Web/CSS/grid) 布局结合而成，让你轻松快速搭建项目雏形。



### 切图铺助
